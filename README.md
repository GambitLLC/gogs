## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```
   git clone https://github.com/GambitLLC/gogs
   ```
2. Install Go modules
   ```sh
   go get
   ```
3. Build the project
   ```sh
   go build
   ```
